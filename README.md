# 📊 Tech Job Market Analysis & Salary Insights

## 🚀 Project Overview

This project analyzes tech job market trends, focusing on salary distribution, job demand, and the impact of experience and location.

---

## 🛠 Tools Used

* Python (Jupyter Notebook)
* PostgreSQL (SQL)
* Power BI

---

## 📂 Project Structure

* **data/** → Raw and cleaned datasets
* **notebooks/** → Data cleaning and EDA
* **sql/** → SQL queries for analysis
* **powerbi/** → Dashboard file and screenshots
* **report/** → Final project report

---

## 🔍 Key Insights

* Data-related roles offer higher salaries
* Salary increases with experience
* High-demand roles often have moderate salaries
* Niche roles provide higher compensation
* Remote jobs are increasingly common
